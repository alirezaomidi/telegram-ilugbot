# Isfahan LUG Telegram Bot

### Installing
You should have python installed on a Linux/Mac machine. Then:

```bash
$ git clone https://github.com/alirezaomidi/telegram-ilugbot
$ cd telegram-ilubbot
$ . script.sh
```

### Running
```bash
$ python bot.py
```
